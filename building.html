The definition of anomaly in building data is based on the number of standard deviations from a reference value. The reference pattern is described as follows:

  • The initial 5 minutes interval data were aggregated by hours.
  • The statistical median of each variable is computed as a median value of a variable in a particular zone and hour. Due to the absence of employees during the weekend we distinguished two kinds of references, weekdays and weekends.
  • The standard deviation (SD) for every variable uses the units of the original variable. It is not possible to compare variables that use different unit metrics. Therefore, the SD result is divided by the reference value (‘SD times’).
